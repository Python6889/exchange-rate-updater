# 沪港通结算汇率自动更新程序

这是一个自动更新沪港通结算汇率到飞书表格的程序。

## 功能特点
- 每天自动获取沪港通结算汇率数据
- 自动识别新增数据，避免重复更新
- 通过GitHub Actions实现定时自动化
- 只更新新增数据，保留历史记录

## 使用方法
1. 在飞书开放平台创建应用并获取API凭证
2. 在GitHub仓库的Settings中设置Secrets：
   - APP_ID: 飞书应用的App ID
   - APP_SECRET: 飞书应用的App Secret
   - TABLE_TOKEN: 飞书表格的Token
   - SHEET_ID: 工作表的ID
3. 程序会在每天0点（北京时间）自动运行

## 依赖库
- akshare: 用于获取沪港通结算汇率数据
- pandas: 数据处理
- requests: HTTP请求库

## 数据流程
1. 从akshare获取最新的结算汇率数据
2. 从飞书表格读取现有数据
3. 比较并识别新增数据
4. 将新增数据追加到飞书表格末尾

## 注意事项
- 首次使用时，表格应为空或只有表头
- 程序会自动添加"更新时间"列
- 如果akshare接口不可用，程序会失败并记录日志
